Health Insurance - Responsive Form Integration

This website was created to provide potential clients with a way to get a health insurance plan quote for their family. The task was to make an appealing form design that would respond to a numerous number of devices, and provide a simple experience that leads the user through the process of getting a quote or submitting an application for health insurance. The form design for the site was required to be highly customizable, and maintain usability during the application process. We chose to use NodeJS and React for this site since both frameworks allow for using the latest developments in Javascript (ES6/ES7).

Paradise Compared - Responsive Design

This website was created to provide responsive design in a layout using HTML5 and experimental CSS3 related features. The page content describes a few vacations I have been able to take over the years (and was the baseline for this site). The content expands horizonatally as the browser widens. The menu and navigation stays-on-top if the browser(view port) is significantly large, otherwise (ie. for mobile devices) it is simply a part of the page. The bottom footer is sticky and will stick firmly to the bottom section of the browser as long as the content does not exceed the browser page height.

uEmploy - WebManager - SOAP Service xsd1 xsd2

This Java web application uses the the Java displaytag library to easily display SessionBean object Lists in a configurable table. A user can Add, Delete, or Search for employees. This web app makes no database calls directly, instead it uses a JAX-WS SOAP service provided by a Java Service Endpoint Interface. This four tier web application makes use of an Apache Webserver, a Tomcat Java Application Server, a Java SEI, and a MYSQL database. A combination of HTML, CSS, JSP, JSTL, EL, displaytag.org, and JAX-WS provide a simple yet powerful interface.

nCase Firewall - Practicum Project

This program was the subject of an intense practicum I completed at BCIT, in order to attempt to provide better security for enterpises engaged in operating SCADA Systems. It was designed to operate on an Intrinsyc PXA270 embedded Linux board (ie. a handheld sized board). The configuration provides a firewall that does not allow any external access. If external access is required a remote client can request to open a port by using a custom encryption channel. In this way access through the firewall to the internal network is only available to those that know the custom encryption/protocol. Access is further monitored by a configuration file that can be as restrictive as required

Micro Loans - Responsive/Mobile Form

This website was an experiment in using Responsive Web Design to allow for any device to follow through a number of screens to complete the application process for a simple small loan. Depending on the size of the device screen a different number of fields are presented, so that the user never needs to scroll through a page to see all the fields. Numerous cross/browser/platform issues were addressed, in order to ensure the user could view some help text and also enter data without having to do anything other than tap a field, enter data, and click a button to proceed to the next screen.
